# Hurst Time Cycle PRO V2

## Overview

Hurst Time Cycle PRO V2 is a professional TradingView Pine Script v6 project for analyzing Hurst market cycles using trading bars, trading days, and trading weeks.

The project is designed with a modular architecture to support:

- Time Intelligence Engine
- Cycle Engine
- Confluence Engine
- Projection Engine
- Dashboard
- Alerts
- Multi-anchor analysis
- Professional visualization
